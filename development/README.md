# Development

> WIP

1. **Clone the 0xGG/crossnote repository** (Recommended to fork your own)

```bash
$ git clone https://github.com/0xGG/crossnote.git
```

2. **Install all necessary packages**

```bash
$ yarn
```

3. **Start the development server**

```bash
$ yarn start
```
